8. Verbruikers en toestellen

8.1 Stopcontacten (Contactdoos)

Typische configuraties:

  • Normale kring: 16A automaat, 2.5mm¬≤ kabel, max 8 stopcontacten
  • Keuken: 20A differentieel, 2.5mm¬≤ of 4mm¬≤, geaard + kinderveilig
  • Badkamer: 30mA differentieel, halfwaterdicht, geaard
  • Kracht: 32A 3-fasig voor zware machines

8.2 Verlichting

Best practices:

  • Gebruik "Lichtcircuit" voor lamp + schakelaar combinatie
  • Gebruik "Lichtpunt" voor enkel de lamp (zonder schakelaar)
  • Gebruik "Schakelaars" voor enkel de schakelaar
  • Typisch: 10A of 16A automaat, 1.5mm¬≤ kabel
  • Max 8 lichtpunten per kring (aanbeveling)

8.4 Verbruiker als kind

Sommige verbruikers kunnen als "kind" onder een contactdoos hangen:

  2. Selecteer de contactdoos

9.5 Highlighting

Visuele feedback in het schema:

  • Hover - Element licht op bij mouse-over
  • Selectie - Geselecteerd element heeft blauwe rand
  • Link lijst-schema - Hover in lijst highlight in schema en vice versa
